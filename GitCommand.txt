將目前資料夾設定為Git repo，即Git倉庫
$ git init

加入使用者電郵
$ git config --global user.email "ex@example.com"

將Readme.txt檔加入版本控制
$ git add Readme.txt
將目前資料夾全部加入版本控制
$ git add -f --all 

將目前的Change commit
$ git commit -m "<你想說的話>"

// 檔案尚未加入過追蹤時使用，即可恢復到檔案尚未加入暫存區
$ git rm --cached hello.py
// 若檔案已經在 repository 內，則使用以下指令
// repository 與 stage 的檔案都會被還原到 HEAD，但 working directory 內的檔案不變
$ git reset HARD <logID>
